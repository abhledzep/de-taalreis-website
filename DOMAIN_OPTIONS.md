## How to Check Availability

### Option 1: TransIP.nl
1. Visit: https://www.transip.nl/domain-name/
2. Enter domain name
3. Check availability
4. Register (€8-12/year)

### Option 2: Mijndomein.nl
1. Visit: https://mijn.host/
2. Search for domain
3. Register if available

### Option 3: Hostnet.nl
1. Visit: https://www.hostnet.nl/domeinnaam
2. Check and register

## Registration Steps

1. **Check availability** on one of the registrars above
2. **Choose hosting** (or use Vercel for free)
3. **Register domain** (€8-15/year typically)
4. **Configure DNS:**
   - For Vercel: Add A and CNAME records
   - Vercel provides the exact DNS settings

## DNS Configuration for Vercel

Once you have a domain, add these DNS records:

```
Type: A
Name: @
Value: 76.76.21.21

Type: CNAME
Name: www
Value: cname.vercel-dns.com
```
